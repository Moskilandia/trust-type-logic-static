# Trust Type Logic Tool (Static HTML)

A fully static, deploy-ready version of the trust recommendation tool built with HTML + JavaScript + Tailwind CSS.

## Deploy Instructions

1. Push this folder to your GitHub repository
2. Import the repo at https://vercel.com/import
3. No build step needed – it deploys instantly as static

## File Structure

- `index.html` – Main app shell
- `app.js` – Trust logic and question flow
